# Interface_Testing
Repositório para testagem com o framework QT.



# Requisitos
## Mínimo: 

- Python versão >= 3.8
- Numpy versão >= 1.2
- Matplotlib versão <= 3.9
- Pyside 6 versão >= 6.2

## Recomendado:

- Python versão = 3.10
- Numpy versão >= 1.26
- Matplotlib versão = 3.9.0
- Pyside 6 versão = 6.4
